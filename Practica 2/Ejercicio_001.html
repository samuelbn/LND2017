Ejercicio 1 Editores de código

1- Descarga e instala un editor en tu equipo. Brackets es una buena opción. Es software libre, tiene muchos plugins y un gran soporte de la comunidad, aunque puedes instalar cualquier otro que te guste. Otras opciones interesantes son Atom y Sublime.

2- Desde el propio editor crea una carpeta que se llame practicas. Dentro de ella otra que se llame practica2.

3- Realiza los ejercicios del tutorial del W3Schools, desde HTML atributtes hasta HTML Paragraphs incluido. Los ejercicios deben estar dentro de una carpeta llamada practica2. Los ficheros deben llamarse ejercicio_001.html, ejercicio_002.html, etc.
